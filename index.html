Growing on the Basics

디자인의 기본기를 바탕으로 신뢰와 소통을 디자인합니다

디자인 기초

저는 디자인의 본질은 ‘기초’에 있다고 믿습니다. 색감, 형태, 여백 같은 기본적인 요소들이 조화롭게 어우러질 때 비로소 안정적인 디자인이 완성된다고 생각합니다. 이러한 기본기를 바탕으로 한 단단한 디자인은 어떤 프로젝트에서도 흔들리지 않는 힘이 됩니다.

디자인 협업

저는 혼자 잘하는 디자이너보다, 함께 성장하는 디자이너가 더 큰 가치를 만든다고 믿습니다. 팀원들과 의견을 나누며, 서로의 시각과 감각을 존중하는 과정에서 새로운 영감이 생깁니다. 협업을 통해 완성되는 결과물은 언제나 혼자서 만든 것보다 풍성하고 깊이 있습니다.

회사에서의 소통

디자인은 결국 ‘사람과의 소통’이라고 생각합니다. 클라이언트, 팀원, 사용자의 입장을 두루 이해하며, 그들의 생각을 시각적으로 전달하는 것이 제 역할이라 여깁니다. 열린 태도와 진심 어린 대화를 통해 더 나은 결과를 만들어가고 싶습니다.

저는 책임감있게 꼼꼼하게 디자인 하는 사람입니다.

디자인의 완성은 ‘세밀한 부분’에서 결정된다고 생각합니다.픽셀 하나, 간격 하나까지 신경 쓰며 깔끔하고 균형 잡힌 화면을 만듭니다.사용자에게 안정감과 신뢰를 주는 디자인을 만드는 것이 제 목표입니다.

Skills & Languages

디자인부터 웹 구현까지 꾸준히 배우며 실무 감각을 쌓고 있습니다.
새로운 도구와 표현 방식을 익히며 더 완성도 높은 결과를 만들기 위해 노력합니다.

  • html5

    HTML을 통해 웹페이지의 기본 구조를 만들 수 있습니다. 콘텐츠를 논리적으로 구분하고 의미를 부여해 접근성 높은 페이지를 설계할 수 있습니다. 디자인 시안을 실제 웹 화면으로 구현하는 기초가 됩니다.

  • CSS

    CSS를 통해 웹페이지의 시각적 스타일을 자유롭게 표현할 수 있습니다. 색상, 폰트, 레이아웃을 조정해 디자인 의도를 정확히 전달할 수 있습니다. 다양한 디바이스에 맞춘 반응형 웹 디자인을 구현할 수 있습니다.

  • JavaScript

    JavaScript를 통해 웹페이지에 생동감 있는 인터랙션을 추가할 수 있습니다. 버튼 클릭, 스크롤, 애니메이션 등 사용자와 상호작용하는 기능을 구현할 수 있습니다. 이를 통해 사용자의 경험을 더욱 풍부하게 만들 수 있습니다.

  • jQuery

    jQuery를 통해 복잡한 JavaScript 기능을 간단하게 구현할 수 있습니다. DOM 조작, 애니메이션, 이벤트 제어를 손쉽게 다룰 수 있습니다. 효율적으로 동적인 웹 페이지를 제작할 수 있습니다.

  • Figma

    Figma를 통해 웹과 모바일 인터페이스를 시각적으로 설계할 수 있습니다. 컴포넌트와 프로토타입 기능을 활용해 사용자 흐름을 쉽게 확인할 수 있습니다. 디자이너와 개발자 간의 협업을 효율적으로 진행할 수 있습니다.

  • Illustrator

    Illustrator를 통해 로고, 아이콘, 인포그래픽 등 다양한 벡터 디자인을 제작할 수 있습니다. 정확한 도형과 선을 활용해 깔끔한 그래픽 작업이 가능합니다. 브랜드 아이덴티티를 시각적으로 표현할 수 있습니다.

  • Photoshop

    Photoshop을 통해 이미지를 보정하고 합성하여 시각적인 완성도를 높일 수 있습니다. 배너, 썸네일, 웹 시안을 자유롭게 제작할 수 있습니다. 사진과 그래픽을 조합해 감각적인 디자인을 완성할 수 있습니다.

  • React

    React를 통해 컴포넌트 단위로 디자인을 구조화할 수 있습니다. Styled-components나 CSS 모듈을 활용해 각 요소별 스타일을 효율적으로 관리할 수 있습니다. 디자인 변경 시에도 유지보수가 쉬운 구조적 스타일 시스템을 만들 수 있습니다.

  • Git / GitHub

    Git과 GitHub를 통해 디자인 및 코드의 버전을 체계적으로 관리할 수 있습니다. 작업 이력을 기록하고 복원하여 효율적인 협업이 가능합니다. 팀 프로젝트에서 함께 작업하며 유지보수를 원활히 진행할 수 있습니다.
